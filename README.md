# Validador de CPF e CNPJ 🇧🇷

Programa em Python que valida CPF e CNPJ brasileiros usando o algoritmo oficial.

## ✨ Funcionalidades

- ✅ Valida os 2 dígitos verificadores do CPF
- ✅ Valida os 2 dígitos verificadores do CNPJ
- ✅ Aceita com ou sem pontuação (., -, /)
- ✅ Interface simples e direta

## 🚀 Como usar

1. Execute o programa:

```bash
python main.py
```
